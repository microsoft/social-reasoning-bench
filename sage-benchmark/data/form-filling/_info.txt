# Form Filling Task Datasets

- `test-form.yaml` -- single form for testing code
- `form-tasks.yaml` -- manually created forms
- `form-tasks-generated-short.yaml` -- converted from datasets/form_filling/form_filling_scenarios.jsonl. Questions are pulled from the jsonl file.
- `form-tasks-generated.yaml` -- cconverted from datasets/form_filling/form_filling_scenarios.jsonl. Questions are parsed from the pydantic objects and longer.
