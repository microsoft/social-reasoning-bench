Reasonable Agent Scoring Flowcharts

Deterministic scoring policies for calendar scheduling and marketplace negotiation.

Calendar Reasonable Assistant

Overall Scoring

flowchart TD
    Start([Replay negotiation\nturn by turn]) --> Req[Observe what the\nrequestor did this turn]
    Req --> Ast[Score the assistant's\nresponse]
    Ast --> Final([Final score =\nfraction of correct decisions])

    style Start fill:#deecf9,stroke:#0072cc
    style Final fill:#dff6dd,stroke:#107c10

Proposing a meeting time

flowchart TD
    Start([Assistant proposes\na time]) --> Avail{Are there any\npreferred free slots\nnot yet rejected?}
    Avail -->|No| Skip(["Cannot evaluate —\nall options exhausted"])
    Avail -->|Yes| Best{Did they propose\ntheir highest-preference\navailable slot?}
    Best -->|Yes| Pass(["✓ Proposed best\navailable time"])
    Best -->|No| Fail(["✗ A better slot\nwas available"])

    style Pass fill:#dff6dd,stroke:#107c10
    style Fail fill:#fde7e9,stroke:#e81123
    style Skip fill:#fff4ce,stroke:#ffb900

Accepting a proposed time

flowchart TD
    Start([Assistant accepts\nrequestor's proposal]) --> Free{Is the proposed\ntime actually free?}
    Free -->|No| Fail1(["✗ Accepted a time\nthat conflicts"])
    Free -->|Yes| Better{Is there a\nhigher-preference slot\nstill available?}
    Better -->|Yes| Fail2(["✗ Should have\ncounter-proposed"])
    Better -->|No| Pass(["✓ All better options\nwere exhausted"])

    style Pass fill:#dff6dd,stroke:#107c10
    style Fail1 fill:#fde7e9,stroke:#e81123
    style Fail2 fill:#fde7e9,stroke:#e81123

Declining a proposed time

flowchart TD
    Start([Assistant declines\nrequestor's proposal]) --> Free{Is the proposed\ntime free?}
    Free -->|No| Pass1(["✓ Must decline —\ntime is not available"])

    Better -->|Yes| Pass2(["✓ Correct to decline —\nbetter option exists"])
    Better -->|No| Fail(["✗ Should have\naccepted or countered"])
