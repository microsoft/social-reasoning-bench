# Introduction

## Motivation

LLM agents are increasingly deployed in multi-agent settings where social reasoning determines success and failure.

## Problem Statement

Current benchmarks evaluate agents in isolation and cannot measure how they handle privacy, persuasion, and conflicting interests.

## Contributions

SRBench provides three multi-agent environments, four evaluation dimensions beyond task completion, and an adversarial framework for stress-testing social reasoning.
