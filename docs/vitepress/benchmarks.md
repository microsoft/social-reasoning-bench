# Benchmarks

SAGE evaluates LLM agents in three multi-agent environments with information asymmetry.

## Calendar Scheduling

An assistant schedules meetings while protecting private calendar events from a potentially adversarial requestor.

## Form Filling

An assistant fills forms from sensitive artifacts while an interviewer probes for information.

## Marketplace

A buyer and seller negotiate price while each hides a reservation price.

## Adversarial Variants

Every benchmark includes hand-crafted and whimsical adversarial variants targeting privacy, duty of care, and due diligence.
